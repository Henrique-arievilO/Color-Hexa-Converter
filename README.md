# 🎨 Color Hex Converter

A minimalist RGB to Hexadecimal converter built with HTML, CSS and vanilla JavaScript.

This project allows users to convert RGB color values into hexadecimal format with real-time visual feedback, input validation, dynamic interface states, and accessible interaction flow.

---

## ✨ Features

- RGB to Hexadecimal conversion
- Input validation (0–255 range)
- Empty field validation
- Dynamic feedback messages
- Real-time color preview
- Interface state management
- Input locking after conversion
- Reset/clear functionality
- Responsive and minimalist UI

---

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript
